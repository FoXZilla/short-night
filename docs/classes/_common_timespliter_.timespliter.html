Class TimeSpliter

Split a date scope to node by node.

example

new TimeSpliter('2017-01-01 00:00:01', '2019-01-01 00:00:01').split(DateBy.Year)

[ '2018-1-1 00:00:00', '2019-1-1 00:00:00' ]
